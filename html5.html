History

Tim Berners-Lee invented = > www in 1989 1990s.From 1991 to 1998, HTML developed from version 1 to version 4. World Wide Web Consortium (W3C) recommended XHTML 1.0. WHATWG (the Web Hypertext Application Technology Group) was formed in response to slow W3C development, and W3C's decision to close down the development of HTML, in favor of XHTML. In 2006, W3C announced that they would support WHATWG. In 2008, the first HTML5 public draft was released In 2012, WHATWG and W3C decided on a separation: HTML5 Browser Support We can teach old browsers to handle HTML5 Define HTML5 Elements as Block Elements HTML5 defines 8 new semantic HTML elements.
